# celeste-maker
A map editor for CELESTE Classic built in love2d using [picolove](https://github.com/gamax92/picolove).

| Control  | Action |
| -------- | ------ |
| Ctrl+S   | Save cart |
| P        | Switch between playtesting and editing modes |
| S / F    | Move level left and right |
| Right click | Erase |
| Middle click | Sample |
